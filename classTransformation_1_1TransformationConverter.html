◆ SparseAssignments

template<typename T = float>
typedef std::array<Assignment, DIM_3D> Transformation::TransformationConverter< T >::SparseAssignments

Sparse representation of the 3x3 transformation logic.

Contains three assignments, one for each basis vector (Right, Forward, Up). This allows the library to perform transformations using sparse assignments rather than dense matrix multiplications, providing maximum efficiency.
